# Installers

Reactive Trader速 supports multiple desktop platforms, including [OpenFin](https://www.openfin.co/), now known by the brand name **Here&trade;** and [Finsemble](https://documentation.finsemble.com/) now known under the product name [io.connect](https://interop.io/products/io-connect/).

Below you can find links to the installers for the various platforms.
